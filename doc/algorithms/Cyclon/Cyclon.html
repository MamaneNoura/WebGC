Cyclon

This class implements the goosip-base algorithm Cyclon. The local view is basically a set of ID's, each ID identifies a remote peer. Each key in the view GossipProtocol.view is the ID of a remote peer and each key points to an object that contains an age field (timestamp) and a data field (application dependent).
