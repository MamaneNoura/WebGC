<constant> configurationObj :Object

This is an example of a configuration object. Basically, there are two parts in this object: the first part contains properties for the Peer class of PeerJS in the property peerServer and the second part contains an array of gossip protocols in the property protocols; the protocols in the array are going to build clusters with the peers that uses this configuration object.
