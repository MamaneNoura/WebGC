Class: SamplingService

SamplingService

new SamplingService(opts)

This class implements the Random Peer Sampling (RPS) service. The local view is basically a set of ID's, each ID identifies a remote peer.The implementation of the view GossipProtocol.view is an object where the ID of a remote peer is a key and each key points to an Item that contains an age field (timestamp) and a data field (application dependent).
