Class: Cyclon

Cyclon

new Cyclon(algOpts, log, gossipUtil)

Implementation of the gossip-based protocol Cyclon. The local view is an object where each of its keys identify a remote peer (peer ID); the value of each key points to a vector with two entries, the first one is an integer (age of the vector) and the second one is the data owned by the remote peer.
Parameters:
Name Type Description
algOpts Object with the settings of the protocol (fanout, view size, etc.)
log Logger object register any error, warning or info message
gossipUtil GossipUtil object that contains common functions used by gossip protocols
