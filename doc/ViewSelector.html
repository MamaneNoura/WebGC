Class: ViewSelector

ViewSelector

new ViewSelector(profile, log, simFunc)

Ranks items in a gossip view according to a similarity function, this function evaluates to which extend two peer profiles differs from each other.
Parameters:
Name Type Description
profile Profile of the local peer
log Logger (see LoggerForWebWorker) to monitor the actions of the ViewSelector
