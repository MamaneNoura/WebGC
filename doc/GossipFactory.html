Class: GossipFactory

GossipFactory

new GossipFactory(opts)

Implementation of the factory pattern to create instances of gossip protocols, the settings of each protocol are defined in the configuration object. Given that the computation of gossip cycles could takes considerable time, each gossip protocol will be instantiated in the context of a web worker; in that way it is expected to avoid freezing the main thread of JavaScript.

<static> setDependencies()

In some cases, there are gossip protocols that have dependencies among them. This method reads the property dependencies in the configuration object and establishes those dependencies. For this method, a dependency is to share the property of one gossip protocol with another gossip protocol.
